# 📌 Sistema de Información para la Gestión de Clientes y Productos en PYMES

## 📖 Descripción
Aplicación web para la gestión de clientes, productos, ventas y detalle de ventas en pequeñas y medianas empresas de comercio local.  
Incluye autenticación de usuarios, dashboard con métricas y CRUD completo conectado a Supabase.

---

## 🚀 Tecnologías usadas
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **Base de datos:** Supabase (PostgreSQL en la nube)
- **Control de versiones:** GitHub
